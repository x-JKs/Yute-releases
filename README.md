## Features

- **Activity Tracker** - Track raid, dungeon, and other activity completions across all characters in real time
- **In-Game Overlay** - Lightweight overlay showing activity timers, fireteam info, and module status
- **Packet Interception** - Intercept and interact with network packets via WinDivert
- **Personal Bests** - Automatically tracks your fastest clears per activity
- **Activity Toasts** - Desktop notifications when activities complete, with duration and completion status
- **Discord Rich Presence** - Show your current activity on your Discord profile
- **AutoHotkey Integration** - Launch and manage AHK scripts alongside the app
- **Customizable Modules** - Enable/disable features with per-module keybinds and settings

## Requirements

- Windows 10 (build 17763+) or Windows 11
- .NET 8.0 Desktop Runtime ([download](https://dotnet.microsoft.com/download/dotnet/8.0))
- A valid license key

## Installation

1. Download the latest release from the [Releases](https://github.com/x-JKs/Yute-releases/releases/latest) page
2. Extract the zip to a folder of your choice
3. Run `Yute.exe`
4. Enter your license key when prompted

## License Keys

## Auto Updates

Yute checks for updates on every launch. When a new version is available, you'll be prompted to download it. Updates are verified with SHA256 checksums to ensure integrity.

## FAQ

**The app says "No internet connection"**
Yute needs internet access to validate your license and connect to the Bungie API. Check your firewall/antivirus settings.

**My key says "already in use on another machine"**
Keys are hardware-locked to the first machine that activates them. Contact x-JKs if you need your key reset for a new PC.

**The overlay isn't showing**
Make sure Destiny 2 is running and the overlay is enabled in Settings > Overlay. Check that the overlay offset values aren't pushing it off-screen.
